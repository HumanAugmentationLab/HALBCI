To demo the Muse open/closed eye paradigm, run...
1. GUI_1 for extrinisic stimuli or keyboardsetupforIntrinsicEyes for intrinsic stimuli
2. Train_2 to train the model on a CSP paradigm
3. TestTheModelLive_pre4 or Live_3 to stream an output with the model for live classification
4. Neato_4 to use this stream to control a Neato

(Most of) the other files in this folder referenced in the above programs, or are previous versions
